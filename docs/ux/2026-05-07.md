## 🟡 **MEDIUM: Lang="es" pero probablemente el app está en inglés**

**Issue:**
- Declaras `<html lang="es">` pero si tu React app carga en inglés por defecto → mismatch
- Screen readers + SEO se confunden
- LATAM mid-market mezcla spanglish (ej: "onboarding", "churn", "runbook")

**Fix:**
```html
<!-- Si app es español puro -->
<html lang="es-CL">

<!-- Si app es spanglish real (honesto con tu ICP) -->
<html lang="es-419" data-tone="professional-latam">

<!-- En tu React root, asegura que el Provider del i18n matchee -->
```

```tsx
// App.tsx
import { useEffect } from 'react';

function App() {
  useEffect(() => {
    // Sincroniza con <html>
    document.documentElement.lang = 'es-CL';
  }, []);
  
  return <YourRoutes />;
}
```

**Por qué importa:** Google rankea mejor contenido que matchea el lang declarado. Y un Head de Ops chileno/mexicano espera UI en español pero términos técnicos en inglés (es su realidad).

## 🟡 **MEDIUM: Zero preload/prefetch = slow perceived performance**

**Issue:**
- Tu CSS está en `/assets/index-ATNeK4zA.css` pero no hay preload
- Probablemente hay fonts custom sin `font-display: swap`
- Primera carga se siente lenta (aunque no lo sea)

**Fix:**
```html
<head>
  <!-- Preload critical CSS -->
  <link rel="preload" href="/assets/index-ATNeK4zA.css" as="style">
  <link rel="stylesheet" href="/assets/index-ATNeK4zA.css">
  
  <!-- Si usas custom fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
  
  <!-- Prefetch si tienes hero video/imagen pesada -->
  <link rel="prefetch" href="/hero-visual.webp" as="image">
  
  <!-- DNS prefetch para analytics/tracking -->
  <link rel="dns-prefetch" href="https://www.googletagmanager.com">
</head>
```

**Por qué importa:** Un Head de Ops abre tu landing desde un café con WiFi choto. Si tarda >2s en verse algo, asume que tu producto es lento.


**Por qué importa:** Detalle pequeño pero profesional. Un Head de Ops revisa desde su iPhone mientras espera el Metro → tu morado en la barra = consistencia de marca.
