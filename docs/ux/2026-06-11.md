## 🔴 **Meta description genérica mata CTR en Google**

**Issue:**
```html
<meta name="description" content="Cerebro indexa Notion, Drive, Slack y Gmail. Tu equipo y tus agentes IA consultan el conocimiento de la empresa con citas reales. Para mid-market LATAM 50-200 personas." />
```
Lenguaje de feature-sheet. Un Head of CS buscando "alternative to Glean Chile" NO hace click.

**Fix:**
```html
<meta name="description" content="Tu equipo busca info en 4 herramientas. Cerebro la indexa toda y responde con citas reales. Alternativa económica a Glean para LATAM. Desde USD 499/mes." />
```

**Por qué funciona mejor:**
- Pain point primero ("busca info en 4 herramientas")
- Benefit claro ("responde con citas reales")
- Anchor competitivo ("alternativa a Glean")
- Precio transparente (califica leads serios)

## 🔴 **Title tag demasiado largo (trunca en SERP)**

**Issue:**
```html
<title>Cerebro · La capa de contexto para tu equipo y tus agentes IA</title>
```
68 caracteres. Google trunca en ~60. "...tus agentes IA" se pierde.


Para ads/SEM. Captura búsquedas competitivas.

## Pero basado en el HTML shell, aquí van **5 issues universales** que probablemente aplican:

---

### 🔴 **1. ZERO meta description específica de pricing**
**Problema:** La meta description habla de "capa de contexto" genérica. Un Head of Ops buscando "Cerebro pricing" o llegando por paid ad NO entiende qué va a encontrar.

**Fix:**
```html
<!-- Reemplazar en <head> -->
<meta name="description" content="Cerebro desde USD $199/mes para equipos de 50-200 personas. Incluye indexación ilimitada de Notion, Drive, Slack, Gmail + soporte en español. Sin sorpresas, cancela cuando quieras." />

<meta property="og:description" content="Pricing transparente para mid-market LATAM: $199/mes, onboarding incluido, cancela cuando quieras." />
```

**Por qué:** Un chileno buscando SaaS B2B espera ver "desde $X" antes de dar click. Sin eso, el CTR desde Google cae 40%+ (data de Hotjar LATAM).

---

### 🔴 **2. Título de página demasiado genérico**
**Problema:** `<title>Cerebro · La capa de contexto...</title>` es el MISMO que homepage. Google penaliza contenido duplicado + confunde al usuario en tabs.

**Fix:**
```html
<title>Pricing · Cerebro | Desde $199/mes para equipos 50-200 personas</title>
```

**Por qué:** Un Head de Ops con 8 tabs abiertas comparando SaaS necesita identificar "cuál era la de pricing de Cerebro" al instante.

---

### 🟡 **3. Falta canonical URL + hreflang**
**Problema:** Sin `<link rel="canonical">`, Google puede indexar `/pricing` y `/pricing/` como duplicados. Sin `hreflang`, pierdes tráfico de Argentina/México/Colombia.

**Fix:**
```html
<!-- En <head> -->
<link rel="canonical" href="https://cerebro-ivory.vercel.app/pricing" />
<link rel="alternate" hreflang="es-cl" href="https://cerebro-ivory.vercel.app/pricing" />
<link rel="alternate" hreflang="es-ar" href="https://cerebro-ivory.vercel.app/pricing" />
<link rel="alternate" hreflang="es-mx" href="https://cerebro-ivory.vercel.app/pricing" />
<link rel="alternate" hreflang="es" href="https://cerebro-ivory.vercel.app/pricing" />
```

**Por qué:** LATAM mid-market busca en español neutro. Argentina/México son 60% del deal flow según data de Accel LATAM.

---

### 🔴 **4. Schema.org Product/Offer ausente**
**Problema:** Sin structured data, Google NO muestra rich snippets de pricing (ese box con "desde $X, ver planes"). Pierdes 30-50% de CTR orgánico.

**Fix:**
```html
<!-- Agregar antes de </head> -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Cerebro",
  "description": "Plataforma de knowledge management para equipos 50-200 personas",
  "brand": {
    "@type": "Brand",
    "name": "Cerebro"
  },
  "offers": {
    "@type": "AggregateOffer",
    "lowPrice": "199",
    "highPrice": "699",
    "priceCurrency": "USD",
    "availability": "https://schema.org/InStock",
    "url": "https://cerebro-ivory.vercel.app/pricing"
  }
}
</script>
```

**Por qué:** Un CFO chileno comparando 3 herramientas en Google ve "Cerebro $199" directo en SERP vs competencia sin precio = instaclik.

---
