## 🔴 **1. Hero invisible – No hay H1 ni propuesta de valor above the fold**

**Problema:**  
El HTML renderizado es un React shell vacío. Sin JS habilitado (crawlers, usuarios con conexiones lentas), la página está **en blanco**. Ni siquiera hay un H1 semántico para SEO.

- Google no indexa correctamente (no ve contenido principal)
- Un Head de Ops que llegue en 3G mobile ve pantalla blanca por 2-3 segundos
- Bounce rate explota

## 🔴 **2. Meta description genérica – No habla al dolor del ICP**

**Problema:**  
```html
<meta name="description" content="Cerebro indexa Notion, Drive, Slack y Gmail. Tu equipo y tus agentes IA consultan el conocimiento de la empresa con citas reales. Para mid-market LATAM 50-200 personas." />
```

Un Head de Ops en LinkedIn ve esto y piensa: *"Ah, otro Notion AI"*. No conecta con el **dolor real**: pérdida de tiempo buscando info dispersa, onboarding lento, preguntas repetidas a Slack.

**Fix:**

```html
<meta name="description" content="¿Tu equipo pierde 2 horas al día buscando info en Notion, Drive y Slack? Cerebro indexa todo y responde con citas reales. Para Heads de Ops/CS en LATAM mid-market. 1/10 del costo de Glean." />
```

**También en OG:**

```html
<meta property="og:description" content="Tu equipo pierde 2 hrs/día buscando docs. Cerebro indexa Notion + Drive + Slack + Gmail. Pregunta en lenguaje natural, respuestas con citas. Para Ops/CS LATAM mid-market. Desde USD 290/mes." />
```

---

## 🔴 **3. Keywords irrelevantes – No posicionan para búsquedas reales**

**Problema:**  
```html
<meta name="keywords" content="knowledge base, IA empresarial, RAG, MCP server, Glean alternativa, Notion AI alternativa, LATAM SaaS, B2B Chile" />
```

- "RAG" y "MCP server" son jerga técnica que un Head de Ops NO busca en Google
- "LATAM SaaS" es demasiado broad
- Faltan long-tail keywords que SÍ buscan: "buscar información empresa Slack", "onboarding empleados automatizado", "alternativa Glean barata Chile"

**Fix:**

```html
<meta name="keywords" content="alternativa Glean barata, Notion AI para empresas Chile, buscar información Slack Gmail, knowledge base América Latina, onboarding empleados automatizado, Customer Success herramientas LATAM, Ops tools mid-market" />
```

**Mejor aún:** eliminar `<meta name="keywords">` (Google lo ignora desde 2009) y enfocar en:
- Contenido semántico (H1, H2, strong)
- Schema.org (que está vacío, ver punto 5)

---

## 🟡 **4. Favicon emoji – No funciona en todos los navegadores**

**Problema:**  
```html
<link rel="icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>🧠</text></svg>" />
```

- Safari iOS < 15 no renderiza emojis en SVG data URIs
- Windows Chrome a veces muestra un cuadrado gris
- No hay fallback PNG/ICO

**Fix:**

1. **Genera un favicon proper:**

```bash
# Usa https://favicon.io/favicon-generator/ o Figma
# Exporta 32x32 PNG con fondo transparente del emoji 🧠
```

2. **Agrega al `<head>`:**

```html
<!-- Favicon moderno (SVG) -->
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />

<!-- Fallback PNG (Safari viejo, Android) -->
<link rel="icon" type="image/png" sizes="32x32"

## Pricing (`/pricing`)

# 🧠 Cerebro Pricing Page Audit

## Issues priorizados

### 🔴 HIGH #1: **La página no existe (React SPA sin render)**
**Problema:** El HTML es solo el shell. El contenido de pricing se renderiza en client-side React. Un Head de Ops revisando pricing en móvil con 3G lenta ve pantalla en blanco 3-5 segundos. Google tampoco indexa el contenido real.

**Fix:** SSR o pre-render estático.
```bash
# Si usas Vite + React Router
npm install -D vite-plugin-ssr
# O migrar a Next.js/Remix para SSG automático
```

**Alternativa rápida:** Meta tags de pricing en el `<head>`:
```html
<meta property="og:title" content="Cerebro Pricing: desde $X/mes para equipos 50-200 personas" />
<meta name="description" content="Plan Startup $X, Plan Scale $Y. Incluye indexación ilimitada de Notion, Drive, Slack, Gmail. 14 días gratis, sin tarjeta." />
```

---

### 🔴 HIGH #2: **Falta el CTA principal en el `<head>` (pre-render)**
**Problema:** No hay estructura semántica de pricing. Usuarios y crawlers no ven beneficios antes de que JS cargue.

**Fix:** Schema.org `<script type="application/ld+json">` en el HTML base:
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Cerebro Knowledge Platform",
  "description": "Capa de contexto para equipos y agentes IA en LATAM",
  "offers": [
    {
      "@type": "Offer",
      "name": "Plan Startup",
      "price": "299",
      "priceCurrency": "USD",
      "availability": "https://schema.org/InStock",
      "eligibleRegion": "CL"
    },
    {
      "@type": "Offer",
      "name": "Plan Scale",
      "price": "799",
      "priceCurrency": "USD"
    }
  ]
}
</script>
```

---

### 🔴 HIGH #3: **Meta description genérica, no habla de precio**
**Problema actual:**
```html
<meta name="description" content="Cerebro indexa Notion, Drive... Para mid-market LATAM 50-200 personas." />
```
Un Head de Ops buscando "cuánto cuesta Glean alternativa Chile" no hace click. Falta **precio ancla** y **diferenciador**.

**Fix:**
```html
<meta name="description" content="Desde USD $299/mes para 50 usuarios. 1/10 del costo de Glean. Incluye indexación ilimitada + agentes IA. 14 días gratis, cancela cuando quieras. LATAM mid-market." />
```

---

### 🟡 MEDIUM #4: **OG image genérica (`og-image.png`)**
**Problema:** La imagen social no comunica pricing. Cuando un Head comparte la URL en Slack del equipo, debería verse:
- Precio destacado
- "vs Glean: 1/10 del costo"
- Logo Cerebro + claim

**Fix:** Generar `og-image-pricing.png` (1200×630):
```html
<meta property="og:image" content="https://cerebro-ivory.vercel.app/og-image-pricing.png" />
```

**Contenido sugerido del PNG:**
```
┌─────────────────────────────────────┐

```

Usa [Satori](https://github.com/vercel/satori) o Figma → export PNG.

---

### 🟡 MEDIUM #5: **Favicon emoji 🧠 se pixelea en tabs oscuros**
**Problema:** El SVG inline con emoji depende del OS. En Windows/Linux puede verse borroso. Un detalle, pero afecta brand perception cuando tienen 20 tabs abiertas comparando tools.

**Fix:** Reemplazar por SVG custom o PNG 32×32 optimizado:
```html
<link rel="icon" type="image/png" href="/favicon-32x32.png" />
<link rel="icon" type="image/svg+xml" href="/cerebro-icon.svg" />
```

**`cerebro-icon.svg`** minimalista:

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32">

(Representación abstracta de conexiones neuronales, funciona en cualquier tema)

**Fix esperado en React:**

```tsx
// ❌ MAL
<label htmlFor="email">Email</label>
<button>Sign up</button>

  htmlFor="email"

>
  Correo electrónico
</label>

  id="email"
  type="email"
  autoComplete="email"
  required
  placeholder="tu@empresa.cl"
  className="w-full px-4 py-3 border border-slate-300 rounded-lg focus:ring-2 focus:ring-indigo-500 focus:border-transparent"

/>
<p id="email-hint" className="text-xs text-slate-500 mt-1">
  Usaremos este correo para enviarte el link de verificación
</p>

  type="submit"

  Crear cuenta gratis

```
