## Landing (`/`)

# 🔴 Audit: Landing Cerebro

## 🔴 HIGH PRIORITY

### 1. **Hero vacío = bounce inmediato**
**Issue:** La landing carga un `<div id="root"></div>` vacío. En conexiones lentas (común en LATAM), el usuario ve pantalla blanca 1-3 segundos. **Perdés al Head de Ops antes de que cargue React.**

**Fix:** Server-Side Rendering (Next.js) o critical HTML inline.

```jsx
// Migrar a Next.js App Router (recomendado)
// app/page.tsx
export default function Home() {
  return (
    <main className="min-h-screen bg-white">
      {/* Hero renderizado desde servidor */}
      <section className="px-6 py-20 max-w-5xl mx-auto">
        <h1 className="text-5xl font-bold text-gray-900 leading-tight">
          Deja de buscar info en 4 herramientas.<br/>
          Pregúntale a Cerebro.
        </h1>
        <p className="mt-4 text-xl text-gray-600 max-w-2xl">
          La capa de contexto operacional para humanos y agentes IA. 
          Indexa Notion, Drive, Slack, Gmail.
        </p>
        <a 
          href="https://cal.com/cerebro/demo" 
          className="mt-6 inline-flex px-6 py-3 bg-indigo-600 text-white font-semibold rounded-lg hover:bg-indigo-700 transition"
        >
          Agendar demo → 15 min
        </a>
      </section>
    </main>
  );
}
```

**Alternativa rápida (sin migrar):**
```html
<!-- En index.html, dentro de <div id="root"> -->
<div id="root">
  <div style="max-width:1200px;margin:0 auto;padding:80px 24px;font-family:system-ui">
    <h1 style="font-size:48px;font-weight:700;line-height:1.2;color:#111">
      Deja de buscar info en 4 herramientas.<br>Pregúntale a Cerebro.
    </h1>
    <p style="font-size:20px;color:#666;margin-top:16px">
      Cargando...
    </p>
  </div>
</div>
```

### 2. **Meta tags prometen "MCP server" pero un Head de Ops NO sabe qué es eso**
**Issue:** Keywords `"MCP server, RAG, Glean alternativa"` son engineer-speak. Tu ICP (Head de Ops en Chile) piensa en *"buscar documentos más rápido"*, no en arquitectura.

**Fix:** Humanizar meta description.

```html
<!-- ❌ ANTES -->
<meta name="description" content="Cerebro indexa Notion, Drive, Slack y Gmail. Tu equipo y tus agentes IA consultan el conocimiento de la empresa con citas reales. Para mid-market LATAM 50-200 personas." />

<meta name="keywords" content="knowledge base, IA empresarial, RAG, MCP server, Glean alternativa, Notion AI alternativa, LATAM SaaS, B2B Chile" />

<!-- ✅ DESPUÉS -->
<meta name="description" content="Encontrá la info de Notion, Drive, Slack y Gmail en segundos. Sin buscar en 4 tabs. Para equipos de 50-200 personas en LATAM. Prueba gratis 14 días." />

<meta name="keywords" content="buscar documentos empresa, Notion search, alternativa Glean Chile, knowledge base LATAM, IA para equipos, búsqueda inteligente documentos" />
```

**Bonus:** El OG title actual está perfecto (`"Deja de buscar info en 4 herramientas"`), pero la description técnica lo contradice.

### 4. **Favicon emoji puede fallar en navegadores antiguos (Edge legacy, Safari 13)**
**Issue:** Usás `data:image/svg+xml` con emoji. En Windows corporativo (común en mid-market LATAM) puede mostrarse roto.

**Fix:** Generar PNG real con emoji renderizado.

```html
<!-- Crear cerebro.png 32x32 y 180x180 -->
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png" />
<link rel="apple-touch-icon" sizes="180x180" href="/favicon-180.png" />

<!-- Fallback SVG para browsers modernos -->
<link rel="icon" type="image/svg+xml" href="/cerebro.svg" />
```

**Archivo `/public/cerebro.svg`:**

```tsx
// App.tsx
```

```tsx
const Pricing = lazy(() => import('./pages/Pricing'))
```

## 🟡 **3. Falta breadcrumb navigation**

**Problema:**

**Fix:**

```jsx
// components/Breadcrumb.tsx
export function Breadcrumb({ items }) {
  return (
    <nav aria-label="Breadcrumb" className="mb-8">
      <ol className="flex items-center gap-2 text-sm text-slate-500">
        {items.map((item, i) => (
          <li key={i} className="flex items-center gap-2">
            {i > 0 && <span className="text-slate-300">/</span>}
            {item.href ? (
              <a href={item.href} className="hover:text-slate-900 transition-colors">
                {item.label}
              </a>
            ) : (
              <span className="text-slate-900 font-medium">{item.label}</span>
            )}
          </li>
        ))}
      </ol>
    </nav>
  );
}
```
