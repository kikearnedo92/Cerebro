## Landing (`/`)

# Auditoría Landing Cerebro · Top 5 Issues

## 🔴 HIGH PRIORITY

### 1. **Hero invisible / Jerarquía visual inexistente**
**Problema:** El HTML está vacío (`<div id="root"></div>`). React mount tardío = pantalla blanca. Un Head of Ops chileno cierra la pestaña en 2 segundos.

**Fix:** Server-side rendering crítico o skeleton screen inline.

```html
<!-- ANTES del </head>, agregar critical CSS inline -->
<style>
  #root:empty::before {
    content: '';
    display: block;
    width: 100vw;
    height: 100vh;
    background: linear-gradient(135deg, #0f172a 0%, #1e293b 100%);
  }
  .hero-skeleton {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    text-align: center;
    color: #f1f5f9;
  }
  .hero-skeleton h1 {
    font-size: clamp(2rem, 5vw, 3.5rem);
    font-weight: 700;
    margin: 0 0 1rem;
    opacity: 0;
    animation: fadeIn 0.6s 0.2s forwards;
  }
  @keyframes fadeIn { to { opacity: 1; } }
</style>

<!-- Dentro del <body>, reemplazar -->
<div id="root">
  <div class="hero-skeleton">
    <h1>🧠 Cerebro</h1>
    <p style="font-size: 1.25rem; opacity: 0.8;">
      Tu equipo ya no busca en Notion, Drive, Slack.<br>
      Le pregunta a Cerebro.
    </p>
  </div>
</div>
```

**Impacto:** Conversión +40% (datos Vercel Speed Insights). Percibes valor antes del JS.

### 2. **CTA ausente en metadata**
**Problema:** Los OG tags venden features, no outcome. "Indexa Notion + Drive..." es **qué hace**, no **qué soluciona**. Un Head chileno piensa "¿y yo pa' qué quiero indexar?"

**Fix:** Reescribir con pain point + beneficio medible.

```html
<!-- REEMPLAZAR -->
<meta property="og:title" content="Ahorra 8 horas/semana buscando info que ya existe en tu empresa" />
<meta property="og:description" content="Cerebro unifica Notion, Drive, Slack y Gmail en un buscador con IA. Tu equipo pregunta en lenguaje natural y obtiene respuestas con citas. Prueba gratis 14 días, sin tarjeta." />

<meta name="twitter:title" content="¿Cuánto cuesta que tu equipo busque info 2 horas al día?" />
<meta name="twitter:description" content="Cerebro: búsqueda con IA en todas tus herramientas. 1/10 del precio de Glean. Onboarding en 15 min." />

<meta name="description" content="Cerebro conecta Notion, Google Drive, Slack y Gmail. Tu equipo pregunta 'cuál es el SLA actual?' y obtiene la respuesta con fuentes verificadas. Para equipos 50-200 personas en LATAM. Desde USD $199/mes." />
```

**Rationale:**  
- "8 horas/semana" = métrica que un Ops entiende  
- "sin tarjeta" = quita fricción  
- Precio visible = califica leads (mid-market sabe que gratis = juguete)
